About the Project

The ALS Alert System is designed to help late-stage ALS patients communicate urgent needs using eye movement. Unlike expensive eye-tracking systems, this solution is low-cost, simple, and focused on emergency alerts.

The system uses electrooculography (EOG) to detect eye motion and trigger alerts to caregivers via sound and mobile notifications.

Key Features

Eye Movement Detection

Uses EOG sensors to detect intentional left-right eye movement.

Audible Alarm

Triggers a loud alarm to alert caregivers nearby.

Mobile Notifications

Sends alerts to a phone app using cloud communication.

Low Cost

Designed to cost under $100, making it accessible.

Technology Overview

  • EOG Sensors for eye tracking
  • ESP-based microcontroller
  • Signal processing (time + frequency domain)
  • Wi-Fi + Firebase cloud communication
  • Mobile app integration

Android Application

The ALS Alert System includes a companion Android application that enables remote monitoring and control of the device. The app connects to the system via Firebase, allowing caregivers to receive alerts even when they are not nearby.

Real-Time Alerts

Receive instant notifications when the system detects a distress signal.

Alarm History

View timestamps of previous alerts for monitoring and review.

Calibration Controls

Start and manage device calibration directly from the app.

Manual Controls

Trigger test alarms or stop active alarms remotely.
